CMD Commands

mkdir client

  • makes a directory (folder) with the name client

cd subclient

  • change directory to a subfolder with the name subclient

cd ..

  • go one directory higher

ctrl+c

  • stop running the program

npx http-server -o -p 9999

  • npx: Executes a package without installing it globally. It will download and run the package just for this command
  • http-server: The Node.js package that serves files in the current directory
  • -o: Opens the default web browser automatically when the server starts
  • -p 9999: Runs the server on port 9999 instead of the default port 8080
