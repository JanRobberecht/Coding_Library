Install

cd client (for a CRUD app)
cd folder (for only a React app)

npx create-vite

  • Project name:
    • You can type a "." (dot) to keep the name of your folder that you already created. This is recommended by Vite.
    • If you give another name, Vite uses this name to create the folder.
    • If the folder has files already, Vite will give a warning
  • Package name:
    • You have to type the full name of the folder. A "." (dot) is not enough.
    • This is the name that will be stored in your package.json under name.
    • It is the identification of your project for npm, Node.js and package management tools.
    • It must be lowercase without spaces; "-" and "_" are allowed.
  • Select a framework:
  • Select a variant:

npm install

  • This will install the folder "node_modules" in your project folder.

npm run dev

  • Take the localhost url to see the app.

Dependencies (NPM Packages)


Axios

Axios is a promise-based HTTP Client for node.js and the browser. It is isomorphic (= it can run in the browser and nodejs with the same codebase). On the server-side it uses the native node.js http module, while on the client (browser) it uses XMLHttpRequests.
